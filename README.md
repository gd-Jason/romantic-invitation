# Romantic Invitation

一个使用原生 HTML、CSS 和 JavaScript 制作的浪漫约会邀请函。项目采用酒红、奶油白和金色的信件视觉，包含完整的分步邀请流程、动画反馈与隐藏彩蛋，无需安装依赖或执行构建。

## 功能

- 点击信封开启邀请函，伴随花瓣飘落和环境光动画
- “让我想想”按钮会逐渐缩小，“我愿意”按钮会逐渐放大
- 接受邀请后显示惊喜确认页面
- 选择约会日期和时间，支持空值抖动及“小笨蛋，点这里啦 👆”动画提示
- 从粤菜、川湘菜、日韩料理、火锅、烧烤烤肉和甜品饮品中选择餐食
- 支持填写其他想吃的美食
- 最终页展示日期、时间和餐食摘要
- 根据所选日期时间显示实时心动倒计时
- 连续点击最终页心形三次可触发隐藏告白彩蛋
- 彩蛋包含居中秘密信笺、心形印章、金色印记、星光、背景聚焦和庆祝粒子动画
- 彩蛋提供“收好这句话”按钮，关闭后倒计时和约会信息继续保留
- 支持桌面端和移动端响应式布局
- 支持系统“减少动态效果”设置

## 邀请流程

```text
打开信封
  → 接受邀请
  → 惊喜确认
  → 选择日期和时间
  → 选择餐食
  → 查看约会邀请函与倒计时
```

## 本地运行

可以直接打开 `index.html`，也可以在项目目录启动静态服务器：

```powershell
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## 常用自定义

### 修改邀请文案

编辑 `index.html` 中对应页面的标题、说明和按钮文字。

### 修改餐食分类

编辑 `index.html` 中带有 `data-food` 属性的按钮。`data-food` 的值会显示在最终邀请函中。

### 修改可选时间

编辑 `app.js` 中生成时间选项的循环。目前提供 `10:00` 至 `21:00`、每 30 分钟一个选项。

### 修改隐藏彩蛋

编辑 `index.html` 中的 `#secretMessage`，可以修改彩蛋印记、秘密文案和关闭按钮文字。

彩蛋默认需要连续点击最终页中心心形三次触发。触发次数、点击间隔和关闭行为可以在 `app.js` 中的 `secretClicks` 与 `closeSecretMessage()` 相关逻辑里调整。

### 修改心动倒计时

倒计时会自动使用用户选择的日期和时间，并在最终页约会信息框下方实时更新。时间到达后显示“就是现在，准备见面吧 ♥”。相关逻辑位于 `app.js` 的 `startCountdown()`。

## 部署

这是一个纯静态项目，可直接部署到 GitHub Pages、Vercel、Netlify 或其他静态网站托管平台。部署目录选择项目根目录即可，不需要构建命令。
